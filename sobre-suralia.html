El origen del proyecto

Una forma sencilla de encontrar algo diferente

Muchas veces queremos salir, conocer un lugar nuevo o hacer una actividad distinta, pero terminamos buscando durante demasiado tiempo entre redes sociales, páginas dispersas y recomendaciones difíciles de organizar.

Suralia reúne en un mismo espacio rutas, conciertos, actividades culturales, gastronomía, talleres y experiencias para que encontrar un plan sea más rápido, claro y agradable.

El objetivo no es enseñar únicamente los lugares más conocidos, sino también descubrir propuestas pequeñas, planes locales y experiencias que normalmente pasan desapercibidas.

Detrás del proyecto

Desarrollado por Fernando Graciani Aguirre

Suralia es un proyecto web. La plataforma combina diseño, programación y experiencia de usuario para resolver una necesidad cotidiana: encontrar actividades interesantes sin perder tiempo.

El proyecto ha sido desarrollado con HTML, CSS y JavaScript, incorporando funcionalidades como favoritos, reservas, perfiles de usuario, publicación de planes y almacenamiento local.
